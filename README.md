# Analyse des ventes — Python

## Objectif
Analyser un fichier de ventes au format CSV afin d’extraire
des indicateurs clés (KPI) utiles à la prise de décision.

Ce projet a été réalisé dans un objectif d’apprentissage
du métier de Data Analyst.

---

## Données
Le fichier `ventes.csv` contient les informations suivantes :

- ID de commande
- Nom du client
- Pays
- Montant de la commande

---

## Technologies utilisées
- Python 3
- Manipulation de fichiers CSV
- Structures de données (listes, dictionnaires)

---

## Analyses réalisées
- Chiffre d’affaires total
- Nombre de commandes
- Panier moyen
- Ventes par pays
- Ventes par client
- Identification du client le plus rentable

---

```bash
python analysis.py
